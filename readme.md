# Сортировка студентов с использованием паттерна Strategy

## Описание

Консольное приложение для демонстрации различных алгоритмов сортировки студентов.
Реализованы классические сортировки (Bubble, Insertion, Quick, Merge) и специальные версии для дополнительного задания.

## Реализованные требования

- Паттерн Builder для класса `Student` с валидацией
- Кастомная коллекция `CustomArrayList<E>` (полная реализация `List<E>`)
- Паттерн Strategy для сортировки
- Сортировка по всем трём полям (группа, средний балл, номер зачётки)
- Заполнение списка: вручную, случайно, из JSON-файла
- Доп.1: Специальные сортировки — сортируются только студенты с чётным номером зачётки
- Доп.2: Запись в файл в режиме добавления (append)
- Доп.3: Заполнение через стримы
- Доп.4: Многопоточный подсчёт вхождений элемента
- JUnit 5 тесты с хорошим покрытием

## Как запустить

1. Склонируйте репозиторий
2. Откройте в IntelliJ IDEA / Eclipse
3. Запустите класс `com.learn.Main`
4. Следуйте меню

## Структура проекта

- `model` — Student с Builder
- `collection` — CustomArrayList
- `sort` — все стратегии сортировки
- `file_access` — работа с JSON (чтение/добавление)
- `service` — StudentService с логикой
- `Main` — консольное меню

# Team Project Aston

Проект по реализации алгоритмов сортировки с дополнительной функциональностью.

## Структура проекта

```
src/main/java/com/learn/
├── Main.java                    - Главный класс с демонстрацией работы
├── model/
│   └── Student.java             - Класс студента для сортировки
└── sort/
    ├── BubbleSort.java          - Сортировка пузырьком
    ├── QuickSort.java           - Быстрая сортировка
    ├── MergeSort.java           - Сортировка слиянием
    ├── InsertionSort.java       - Сортировка вставками
    ├── SpecialBubbleSort.java   - Модифицированная версия пузырька
    ├── SpecialQuickSort.java    - Модифицированная версия быстрой сортировки
    ├── SpecialMergeSort.java    - Модифицированная версия слияния
    └── SpecialInsertionSort.java - Модифицированная версия вставками
```

## Описание

Базовые алгоритмы сортировки сортируют все элементы списка по возрастанию значения поля `recordBookNumber`.

Специальные алгоритмы сортировки работают следующим образом:

- Элементы с четными значениями поля `recordBookNumber` сортируются в натуральном порядке
- Элементы с нечетными значениями остаются на исходных позициях

## Требования

- Java 17
- Maven

## Запуск

```bash
mvn compile exec:java -Dexec.mainClass="com.learn.Main"
```

Или через IDE:

- Открыть проект в IntelliJ IDEA
- Запустить класс Main
